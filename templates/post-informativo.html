Paróquia Santuário de São Raimundo Nonato

Informativo

Esta página web paróquial, a princípio conterá informações importantes para a comunidade como: horários das celebrações, informativos a respeito de ação pastoral, saber sobre os grupos, informações de maior importância. Este é um site estático, para repassar informações de forma mais rápida. Com dominío gratuíto e sem propagandas.
